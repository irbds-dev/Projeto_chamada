# 📝 Sistema de Chamada Digital - Itapevi (Módulo Complementar)

Este projeto consiste em uma aplicação web desenvolvida para modernizar o processo de registro de frequência em uma escola estadual de **Itapevi, SP**. O sistema atua como uma solução complementar ao ecossistema **Escola do Futuro**, focando na agilidade do preenchimento diário pelos docentes.

---

## 🚀 Principais Funcionalidades

- **Substituição de Planilhas:** Interface otimizada para substituir o preenchimento de células do Excel por seleções intuitivas.
- **Dashboard de Engajamento:** Visualização rápida de turmas com maior índice de evasão.
- **Exportação de Dados:** Geração de relatórios em formatos compatíveis para auditorias e reuniões pedagógicas.
- **Design Responsivo:** Acesso facilitado via smartphone para chamada diretamente em sala de aula.
